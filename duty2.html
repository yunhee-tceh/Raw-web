웹 개발자 (Web Developer)

  1. 직군
  2. 직무
  3. 역량

직무

프론트엔드 (Front End) 개발자
    웹 페이지 화면에서 이용자와 직접 접촉하는 부분에 업무를 담당합니다.
백엔드 (Back End) 개발자
    웹 페이지 화면에서 이용자가 볼 수 없는 부분 (웹 서버 구축, 데이터베이스의 연동, 서버상 API제작 및 개발)과 프론트엔드에서 제작한 웹 기능적으로 동작할 수 있게 하는 업무를 담당합니다.
풀스택 (Fulla stack) 개발자
    웹에 (프론트엔드 (Front End)와 백엔드 (Back End) 대하여 전반적인 업무를 담당합니다.
